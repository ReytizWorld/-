### Установка в Google Chrome
 
(и прочие браузеры на основе Chromium)
 
0. Скачать [архив с кодом](https://github.com/ReytizWorld/-/releases/download/%D0%9E%D1%82%D0%B2%D0%B5%D1%82%D1%8B%40%D0%BB%D0%B0%D0%B9%D0%B2/--main.zip) и распаковать его.
1. Открыть страницу [chrome://extensions](chrome://extensions)
2. Справа-вверху включить "Режим разработчика".
3. Нажать "Загрузить распакованное расширение".
4. Выбрать папку  (их может быть несколько, тогда выбрать самую внутреннюю, а то будет ошибка про файл манифеста)
5. Справа от адресной строки нажмите кнопку "Расширения", выберите "Закрепить". На панели справа появится синяя иконка расширения. Нажав на нее правой кнопкой мыши, можно включить автопостинг ответов (спам), настроить спам вопросов.
6. PROFIT! Зайдите на [otvet.life](https://otvet.life) и другие домены, вы увидите, что плагин работает, например - появились смайлики в поле ввода ответа.

### Что дает плагин
 
1. Вставка emoji в текст.
 
2. Возможность просматривать вопросы\ответы забаненных аккаунтов.

3. Просмотр скрытого содержимого — в уведомлениях

4. Поддержка всех доменов — otvet.life, otvet.live, live-otvet.online, IP-адреса
 
### Безопасность
 
Плагин работает только внутри браузера и только на сайте otvet.life (и другие домены), у него нет никакого доступа ни к каким другим сайтам.

Кроме того, он не подключается ни к каким серверам кроме otvet.life (и других доменов) и не отправляет никуда никакие данные.
